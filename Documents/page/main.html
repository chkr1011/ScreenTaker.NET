Betrieb

Systemvoraussetzungen

Screentaker.NET läuft bisher nur unter den verschiedenen Windows-versionen auf denen das .NET Framework 2.0 von Microsoft enthalten ist. Bei Windows Vista ist es bereits standardmäßig installiert.

Erster Start

Unter Downloads haben Sie drei verschiedene Pakete zur Auswahl. Nachdem Sie sich für eins entschieden, es heruntergeladen und entpackt bzw. installiert haben, erscheint ein Ordner in dem sich unter anderem die ausführbare Datei Screentaker.NET.exe befindet.

Abbildung 1: Icon

Nach einem Doppelklick auf diese Datei startet Screentaker.NET und erscheint als Symbol unten rechts in der Taskleiste. Nun stehen Ihnen verschiedene Möglichkeiten zur Erzeugung eines Screenshots zur Verfügung.

Mit einem einfachen Klick auf das Taskleistensymbol öffnet sich das trübe Auswahlfenster mit dem es Ihnen möglich ist, ein Screenshot eines bestimmten Bereiches zu machen.

Mit dem konfigurierbaren Tastaturkürzel (siehe Tastaturkürzel für die Standardbelegung) können Sie ein Bild des gesamten Bildschirms anfertigen, des aktuell ausgewählten Fensters oder wiederum das Auswahlfenster öffnen.

Mit einem Rechtsklick auf das Taskleistensymbol öffnet sich das Kontextmenü.

Kontextmenü

Abbildung 2: Kontextmenü

Das Kontextmenü unterteilt sich in Über Screentaker.NET, Neuer Screenshot, Screenshot öffnen, Verlauf öffnen, Uploader, Einstellungen, Spenden!, Hilfe, sowie Beenden.

Screentaker.NET

Öffnet das Informationsfenster, welches unter anderem die Updatefunktion beeinhaltet, mit der Sie nach einer neuen Version von Screentaker.NET suchen können.

Abbildung 3: "Info über"

Neuer Screenshot

Hierbei öffnet sich das trübe Auswahlfenster mit dem es Ihnen möglich ist, ein Screenshot eines bestimmten Bereiches zu machen.

Screenshot öffnen

Hier können Sie ein bereits erstelltes Bild für die nachträgliche Bearbeitung im Bearbeitungsfenster öffnen.

Verlauf öffnen

Öffnet das Verlaufsfenster. Siehe Verlaufsfenster.

Uploader

Abbildung 4: Kontextmenü Uploader

Hinter diesem Menüpunkt versteckt sich die Funktion, Bilder an einen der beiden Hostingdienste ImageShack.us bzw. directupload.net hochzuladen.

Nachdem Sie sich für einen der beiden Dienste entschieden haben, werden Sie zunächst aufgefordert ein Bild auszuwählen, welches Sie hochladen möchten. Zum weiteren Vorgehen gehen Sie zum Punkt Uploadfenster.

Einstellungen

Öffnet das Einstellungsfenster. Siehe Einstellungsfenster.

Spenden!

Hier können Sie unsere Entwicklung durch eine kleine Spende unterstützen.

Hilfe

Hier gelangen Sie zu der Online-Dokumention.

Beenden

Hiermit wird Screentaker.NET beendet.
