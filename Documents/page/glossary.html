4. Glossar

7-Zip
Ein frei verfügares Packprogramm, das mit dem 7z-Format eins der besten Formate in punkto Kompression mitbringt. http://www.7-zip.org/de
C#
Eine standardisierte Programmiersprache, in der Screentaker.NET entwickelt wurde. http://de.wikipedia.org/wiki/C-Sharp
GNU General Public License
Eine Lizenz für freie Software, der Sie bei der Benutzung von Screentaker.NET zustimmen müssen.

Originaltext: http://www.fsf.org/licensing/licenses/gpl.html
deutsche inoffizielle Übersetzung: http://www.gnu.de/gpl-ger.html

.NET Framework
Eine Laufzeitumgebung die es unter anderem ermöglicht, in C# entwickelte Programme auszuführen. http://de.wikipedia.org/wiki/Dot_Net

Download: http://msdn.microsoft.com/netframework/downloads/updates/default.aspx
